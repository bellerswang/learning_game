To make these games accessible on May's iPhone, you first need to turn on **GitHub Pages**:

1.  Go to your GitHub repository: [https://github.com/bellerswang/learning_game](https://github.com/bellerswang/learning_game)
2.  Click on the **Settings** tab (top right).
3.  On the left sidebar, click **Pages** (under the "Code and automation" section).
4.  Under **Build and deployment** / **Source**, select **Deploy from a branch**.
5.  Under **Branch**, select **main** and folder **/(root)**.
6.  Click **Save**.

Wait about 1-2 minutes. Refresh the page, and you will see your live URL at the top (e.g., `https://bellerswang.github.io/learning_game/`).
